# Assets Folder

This folder is for additional downloadable files.

## Resume File

**Optional:** Add your resume PDF here.

1. **Create or export your resume as PDF**
   - Use tools like:
     - Google Docs → File → Download → PDF
     - Microsoft Word → Save As → PDF
     - Canva (free resume templates)

2. **Save the file:**
   - Filename: `resume.pdf`
   - Location: `assets/resume.pdf`

3. **Update the link in HTML:**
   - The download link in the About section will automatically work
   - File path: `assets/resume.pdf`
