## âš¡ Common Tasks

### Change Colors
1. Open `css/style.css`
2. Find `:root` section (line ~5)
3. Change `--primary-color: #2563eb;` to your color
4. Save and refresh

### Add a Project
1. Open `index.html`
2. Find Projects section
3. Copy a project card (`<div class="project-card">`)
4. Paste and update content

### Update Skills
1. Open `index.html`
2. Find Skills section
3. Update skill names, descriptions, percentages
4. Change `data-level="90"` to your percentage
